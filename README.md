## ⚙️ Build & Run

### Clone the repository  
```bash
git clone https://github.com/your-username/connect-four.git
cd connect-four
```
### Build
```bash
# Create and enter build directory
mkdir build && cd build

# Generate build files with CMake
cmake ..

# Compile the project
make
```
### Run
```bash
./bin/connect4
```
